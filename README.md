## Getting Started

### Prerequisites

- Git
- Node.js 18+

### Setup and Run

```bash
git clone https://github.com/ENZOMOTIVE/BariChain_prototype.git
cd BariChain_prototype
```

```bash
cd admin-interface
npm install
npm start
```

```bash
npm install
```

```bash
cd user-interface
npm install
npm start
```
